Ch2: Programming Basics

  • Well commented code is the hallmark of a ninja programmer.
  • Two types of comments, // and /**/.
  • JavaScript has similarities with the C programming languages.
  • JavaScript = Series of statements.
  • Each statement ends with a new line or semicolon.
  • Block = Series of statements that are collected together inside curly braces, and do not need to be terminated by a semicolon.
  • Words to not use as a variable in JavaScript: abstract, await, Boolean, break, byte, case, catch, char, class, const, etc.
  • Primitive Data Types: String, Symbol, Number, Boolean, Undefined and Null.
  • Any value that is not primitive data types is an object.
  • Scope: Using const and let to declare variables means they are block scoped.
  • Scope = where a constant or variable is accessible inside the block, they are declared in.
  • Global Scope: any variable declared outside of a block is said to have global scope, that means it is accessible everywhere in the program.
  • Local Scope: That means that any variables defined inside a block using the let or const will only be available inside that block and not be accessible outside of that block.

Ch3: Arrays, Logic, and Loops

  • Arrays are not primitive values but built-in object.
  • We can use heroes to create an empty array.
  • Each item in an array can be treated like a variable.
  • We can create an array literal using square brackets that already contain some initial values
  • The delete operator will remove an item from an array.
  • Destructuring an array is the concept of taking values out of an array and presenting them as individual values. Destructuring allows us to assign multiple values at the same time
  • To remove the last item from an array, we can use the pop() method.
  • The shift() method works in a similar way to the pop() method, but this removes the first item in the array.
  • The push() method appends a new value to the end of the array.
  • The push() method appends a new value to the end of the array.
  • The unshift() method is similar to the push() method, but this appends a new item to the beginning of the array
  • The concat() method can be used to merge an array with one or more arrays.
  • A set can be converted into an array by placing the set, along with the spread operator directly inside an array literal.
  • An empty map object can be created using the new operator and Map() constructor.

Ch4: Functions

  • Function declaration.
  • Function Expressions.
  • A function can also be declared using the constructor function().
  • Invoking a function is to run the code inside the function’s body. To invoke a function, simply enter its name, followed by parentheses.
  • Any parameters a function needs are set when the function is defined. When a function is invoked, it is provided with arguments.
  • Every function has a special variable called arguments.
  • Arguments is not an array.
  • Hoisting is the JavaScript interpreter’s action of moving all variable and function declarations to the top of the current scope, regardless of where they are defined.
  • A function that is passed as an argument to another is known as a callback.
